# Turtle Paint Studios 🎨

Интерактивное приложение для рисования, созданное на стандартной библиотеке Python `turtle`. Проект демонстрирует навыки работы с событиями ввода (клавиатура и мышь) и графическим интерфейсом.

## 🚀 Возможности
- **Кастомный GUI:** Самостоятельно отрисованная панель инструментов с иконками для выбора цветов, толщины кисти и цвета фона.
- **Интерактивные подсказки:** Символьные обозначения над инструментами (К — красный, Ж — желтый, 1-4 — размер и т.д.).
- **Модульная архитектура:** Проект разделен на логику управления (`paint.py`) и визуальный интерфейс (`interface_paint.py`).

## 📁 Структура проекта
- `paint.py` — основной файл: обработка нажатий клавиш и движений мыши.
- `interface_paint.py` — графический движок: отрисовка кнопок, палитры цветов и индикаторов размера кисти.


## ⌨️ Горячие клавиши
- **Стрелки:** Движение черепашки.
- **Цвета:** `R` (красный), `P` (зеленый), `C` (синий), `J` (оранжевый) и др.
- **Фон:** `W` (белый), `B` (черный), `G` (зеленый).
- **Заливка:** `Z` (начать), `X` (закончить).
- **Размер:** клавиши `1`, `2`, `3`, `4`.

## 🛠 Технологии
- **Python 3.x**
- **Turtle Graphics** (Standard Library)

## 🔧 Запуск
Просто запустите скрипт, и откроется окно для рисования:
```bash
python paint.py
```
